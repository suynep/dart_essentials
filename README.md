# Dart Essentials Practice

> Contains solved problems for Week 02 of Internship. 

## Problems 

### ![SOLVED] Project 1: Number List Analyzer
Create a program that:
- Takes a list of numbers as input
- Calculates sum, average, min, max
- Filters odd/even numbers
- Sorts in ascending/descending order
- Removes duplicates

### ![SOLVED] Project 2: Student Grade Manager
Create a program using Maps that:
- Stores student names and grades
- Adds/removes/updates students
- Calculates class average
- Finds top performers
- Filters students by grade range

### Project 3: Text Analyzer
Create a program that:
- Takes a text input
- Counts words, characters, sentences
- Finds word frequency
- Removes common words
- Converts to uppercase/lowercase/title case

### Project 4: Shopping Cart
Create a program that:
- Manages items with name, price, quantity
- Uses Lists and Maps
- Calculates total price
- Applies discounts
- Generates receipt
